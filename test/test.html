Tentukan penyelesaian dari sistem persamaan linear dua variabel berikut dengan metode grafik.

Penyelesaian :

X Y X,Y
X
Y

Petunjuk
Untuk menentukan nilai x, jika y=0. Maka substitusikan y=0 ke persamaan x-y=1
x - y = 1
x - 0 = 1
     x = 1
Sama halnya untuk menentukan nilai y, maka x = 0.
